4-topshiriq: "Bankomat" (Murakkabroq - raise ishlatish)
Foydalanuvchi hisobida 100 000 so'm pul bor deb tasavvur qiling. Undan qancha pul yechishni so'rang.

    Shart: 1. Agar foydalanuvchi 0 dan kichik son kiritsa, ValueError xatosini raise orqali chiqaring.
    2. Agar yechmoqchi bo'lgan summa balansdan ko'p bo'lsa, "Mablag' yetarli emas" degan xabarni chiqaring.
    
    Maqsad: Xatolarni sun'iy ravishda yuzaga keltirish (raise) va shartli tekshirishni o'rganish.
