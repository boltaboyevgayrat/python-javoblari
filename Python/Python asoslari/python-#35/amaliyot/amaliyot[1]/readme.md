1-topshiriq: "Xavfsiz Bo'luvchi" (Oddiy daraja)
Foydalanuvchidan ikkita son so'rang (a va b). Ularni bo'lish natijasini ekranga chiqaring.

    Shart: Foydalanuvchi nol kiritsa (ZeroDivisionError) yoki son o'rniga harf kiritsa (ValueError), dastur to'xtab qolmasdan, tushunarli xato xabarini chiqarsin.

    Maqsad: Bir nechta except bloklari bilan ishlashni o'rganish.
